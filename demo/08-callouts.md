# Callout Boxes

A colored box = a `::: rows` block with colors in the first line + emoji as an icon.

::: rows
{bg=#13241f color=#86efac border=#22c55e}
[[&#10003;|green]] **Success** — everything is working as expected.
:::

::: rows
{bg=#2d1e0f color=#fbbf24 border=#d97706}
[[!|yellow]] **Warning** — caution is advised here.
:::

::: rows
{bg=#2a1215 color=#f87171 border=#ef4444}
[[-|red]] **Error** — something went wrong.
:::
